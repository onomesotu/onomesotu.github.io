SEE MY WORK

Some of my work so far

Armando Pérez is a HTML and CSS static webpage. It incorporates Bootstrap's nav-pills for the navigation and a mini-image gallery using a grid of rows and columns. The images were thumbnails.

Newsroom is a clone for a channel page. Though it is not a web app, it can serve as a prototype.This was a very interesting project because it implemented some useful HTML and CSS techniques learnt on my full-stack development path on codecademy. It was bootstraped. It includes an inline search form, cool colors and miniature newsfeed.

Gameboard is a simple HTML and CSS project that uses the jQuery methods .next(), .children(), toggle() and toggleClass() to show details and give some interactivity

Tribute page to Serena Williams. Gives some details about her careeer, achievements and awards. It was made with HTML and CSS as part of the Free Code Camp course project

The move project involved the use of jQuery UI to design a simple online store product display using an accordion. It also features a jQuery dropdown-menu.
